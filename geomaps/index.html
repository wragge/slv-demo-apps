Explore georeferenced maps from the SLV

About this

Explore georeferenced maps from the State Library of Victoria!

  • click on the ⭠ base map to display a list of georeferenced maps within 50km of your selected point
  • click on one or more of the results to display them layered on the base map
  • click again on a selected result to remove the georeferenced map from the base map
  • click on the layers icon in the top left of the base map to change the order and opacity of the georeferenced maps
  • bookmark or share the url to save your map selection
  • click somewhere else on the base map to start again

This is an experimental interface created by Tim Sherratt during his time as Creative Technologist-in-Residence at the State Library of Victoria Lab.

This interface displays digitised maps from the State Library of Victoria's collection that have been georeferenced by the public using Allmaps.

If you'd like the help georeference the SLV's maps, see the documentation. This blog post has some more information on how all the different parts were put together.
